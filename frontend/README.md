# Optimus - Education Consulting Platform

A comprehensive web application designed to help students explore educational opportunities, select universities and access consulting services for studying abroad.

## 📋 Overview

Optimus is a modern, responsive education consulting platform that simplifies the journey of international education. We provide end-to-end support for students aspiring to study abroad, from course selection to post-admission services.

---

## ✨ Features & Services

### Education Programs
- **Bachelor's Degrees** - Guidance for undergraduate programs worldwide
- **Master's Programs** - Support for postgraduate education pursuits
- **PhD Programs** - Assistance with doctoral degree applications
- **Diploma Courses** - Information on vocational and technical certifications
- **Certifications** - Professional certification course guidance
- **MBBS Abroad** - Specialized support for medical students seeking international education
- **Language Training** - English proficiency and language preparation services

### Comprehensive Consultation Services
- **Country Selection** - Expert guidance on choosing the right study destination
- **University Selection** - Personalized university recommendations based on preferences
- **Course & University Matching** - Intelligent course-university pairing service
- **Admissions Support** - End-to-end admission application assistance
- **Documentation** - Help with visa, SOP and application document preparation
- **Career Counselling** - Post-graduation career planning and guidance
- **Accommodation Services** - Student housing assistance and accommodation booking
- **Education Loan** - Financial planning and education loan guidance
- **Forex Services** - Currency exchange and financial services support

### Additional Features
- **University Directory** - Comprehensive database of universities across study destinations
- **Responsive Design** - Seamless experience across all devices
- **Interactive Navigation** - Intuitive course and service browsing
- **Expert Consultation** - Professional guidance from experienced education consultants

## 🛠️ Technologies Used

This project is built with modern web technologies:

- **Frontend Framework**: React with TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS + PostCSS
- **UI Components**: shadcn/ui
- **Package Manager**: Bun
- **Additional Tools**: ESLint, Prettier

---

## 📦 Installation & Setup

### Prerequisites
- Node.js 16+ or Bun installed
- Git for version control

### Step-by-Step Guide

```bash
# Clone the repository
git clone https://github.com/zoraaisd/optimus

# Navigate to the frontend directory
cd optimus/frontend

# Install dependencies
npm install

# Start the development server
npm run dev
```

The application will be available at `http://localhost:8080` (or the port Vite assigns)
